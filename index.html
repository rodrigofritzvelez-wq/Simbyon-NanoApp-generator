Welcome — Symbion Micro‑Apps

This index bundles five nano‑apps as demo components. Each view is a compact, self‑contained UI you can adapt into a WebView or PWA for distribution.

  • SymbioChain — Ontological immutability ledger and certificate anchoring.
  • SymbioChat — Multi‑agent co‑creative chat (NanoApp).
  • SolarMesh — Energy monitoring + tokenization demo.
  • SymbioMesh — Offline‑first P2P mesh visualizer.
  • Symbiolytics — Coherence scoring & analytics (AMI).

Quick start (dev)

  1. Drop this index.html into your repo root.
  2. Open with Live Server or any static host (GitHub Pages supported).
  3. To convert into a PWA, add a manifest and service worker (see Dev Notes).
